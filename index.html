    Introduction:

    Adventure Works Sales Analysis Dashboard is an Excel-based project showcasing sales performance across regions, products, and time periods. The dashboard provides interactive insights to help management track KPIs, identify growth opportunities, and improve decision-making.

    Project Details:

    In this project, I leveraged advanced Excel features to build an interactive sales analytics dashboard:

    • Collected and cleaned sales data from multiple Excel sources.

    • Used Power Query for data transformation and loading.

    • Applied Power Pivot and created a robust data model with relationships.

    • Designed measures using Lookup functions for KPIs like Revenue, Profit, and Sales Growth.

    • Created interactive visuals including slicers for Region, Product Category, and Year.

    • Designed executive-friendly dashboard layout with charts, trend lines, and summary cards.

    ×

    Introduction:

    Blinkit Grocery Sales Analysis Dashboard is an Excel-based project that provides insights into customer purchase behavior, product demand, and regional sales trends to support operational and marketing strategies.

    Project Details:

    In this project, I applied Excel’s advanced analytics features to analyze Blinkit’s sales performance:

    • Imported and cleaned transactional sales data from Blinkit datasets.

    • Performed data modeling using Power Query and relationships in Power Pivot.

    • Developed measures to track Total Sales, Average Basket Value, and Customer Retention.

    • Designed category-level analysis for Fresh Produce, Packaged Goods, and Beverages.

    • Added slicers for Date, Product Category, and Location to allow drill-down analysis.

    • Created an intuitive dashboard highlighting sales trends and key operational insights.
